To run the tool, you can use a command like this:

python SSRsynth.py --repeat_size 3 --num_microsatellites 1000 --output_file microsatellites.csv

This will generate 1000 synthetic tri-nucleotide microsatellites and save them to a file called microsatellites.csv. You can adjust the repeat_size, num_microsatellites, and output_file parameters to generate microsatellites of different sizes and quantities, and save them to different output files.
